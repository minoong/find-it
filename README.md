### find-it

---

**숙소 예약 서비스는 아래의 기술로 구현 중입니다.**

- React
- Next.js
- Redux Toolkit, Axios, TypeScript
- Google Map API

---

1. 메인화면, 회원가입 모달, 로그인 상태 헤더 구현

2. Common 컴포넌트 구현

   - Button
   - Selector
   - Input
   - Counter
   - Radio

3. Common 폼 검증

4. Geocoding API를 사용하여 구현하였으며, [현재 위치 사용]을 클릭하여 현재 위치를 백엔드에서 데이터를 가져와 세팅합니다.

5. Maps JavaScript API를 사용하여 구현하였으며, 핀을 움직여 정확한 위치를 지정할 수 있도록 합니다.
